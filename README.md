# 🚀 Installation Guide

# 1️⃣ Clone Repository

```bash
git clone https://github.com/Baqirsultan201/HB-EID-APP.git
cd HB-EID-APP
```

---

# 2️⃣ Frontend Setup

```bash
cd frontend
npm install
npx expo start
```

---

# 3️⃣ Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn server:app --reload
```
