# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/Baqirsultan201/HB-EID-APP.git
cd HB-EID-APP
```

---

# 💻 Frontend Setup

```bash
cd frontend
npm install
npx expo start
```

---

# ⚙️ Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn server:app --reload
```

---

# 🔑 Environment Variables

## Frontend `.env`

```env
EXPO_PUBLIC_BACKEND_URL=https://hb-eid-app.onrender.com
```
